1. 프로젝트 개요 - 4 -
연구 목표 및 가설
가. 연구 목표
• Selenium 크롤러를 활용한 한국어 KBO 뉴스 멀티모달 데이터셋 직접 구축(11,531건)
• BERT(klue/bert-base) 기반 감성 3분류(긍정·중립·부정) 및 카테고리 다분류 모델 구현
• ResNet-50 전이학습 기반 썸네일 이미지 감성 분류 모델 구현
• TF-IDF·FISA·BERT 단계적 베이스라인 설정: Late Fusion 앙상블의 성능 우위 정량 검증
• KoBART 기반 KBO 뉴스 자동 요약: Zero-shot 추론 성능 평가
나. 연구 가설
구분 가설 진술
H1 도메인 특화 사전학습(FISA)이 부재한 fine-tuning(BERT)보다 우수한 성능을 보일 것이다.
H2 이미지 모달의 결합(Late Fusion)이 텍스트 단독 대비 Macro-F1 향상을 가져올 것이다.
H3 카테고리 설계의 품질(소수 클래스 통합 여부)이 모델 성능에 직접 영향을 미칠 것이다.
다. 현재까지의 진행 방향
데이터 수집·전처리·자동 라벨링이 완료되었으며, 단일 모달 학습 및 Late Fusion 앙상블의 1차 실험을 종료한 상태
이다. 이어 1차 실험의 한계(특히 카테고리 분류의 추론 비용 누적 문제)를 보완하기 위한 2차 심화 실험(FastCat 구조
)을 추가 진행하였다.

자동 라벨링 및 EDA
가. 감성 라벨링 (키워드 점수 차이 기반)
• 긍정·부정 키워드 사전을 구축하여 본문 등장 빈도의 차이(diff = pos - neg)를 산정
• 임계치(threshold = 2) 적용: diff ≥ 2 긍정, diff ≤ -2 부정, 그 외 중립
• threshold = 1 경계 사례가 전체의 약 22%로 신뢰도 저하: 중립 처리하여 노이즈 감소
감성 건수 비율
중립 5,195건 45.1%
긍정 3,849건 33.4%
부정 2,487건 21.6%
나. 카테고리 라벨링: v1(8개) -> v2(6개) 정제
• 심판(71건) -> 「기타」 편입
• FA + 트레이드 -> 「선수이동」 통합
• 근거: 50건 미만 소수 클래스의 Macro-F1 0 근접 현상 회피
다. 카테고리 ablation 결과(TF-IDF 기준)

4. 모델 설계 - 9 -
텍스트 모달(베이스라인 및 BERT)
가. TF-IDF 베이스라인 (LinearSVC)
• 특징 추출: TfidfVectorizer (max_features=30,000, ngram=(1,2), sublinear_tf=True)
• 분류기: LinearSVC (random_state=42, default C)
• 선정 사유: 학습·추론 모두 CPU 환경에서 수 분 내 종료, 결과 해석 용이
나. FISA; 도메인 특화 사전학습 모델 (추론 전용)
• 뉴스 도메인에 특화된 사전학습 한국어 감성 분석 모델: 별도 학습 없이 추론만 수행
• 도입 의도: 사전학습 모델의 일반화 성능을 KBO 도메인에서 정량 검증 (가설 H1 검증)
다. BERT 본 모델 (klue/bert-base 기반 fine-tuning)
항목 설정값
사전학습 모델 klue/bert-base (한국어 KLUE 벤치마크 기반)
최대 입력 길이 128 토큰
배치 크기 32
Epoch: 감성 / 카테고리 5 / 4
Optimizer · LR AdamW · 2e-5 (linear warmup)
학습 환경 NVIDIA RTX 3090 (RunPod 클라우드)

4. 모델 설계 - 10 -
이미지 모달(ResNet-50 전이학습)
가. 모델 구성
• ImageNet 사전학습 ResNet-50 활용: Backbone 구조 고정
• FC 레이어 교체: 2048 → 512 (ReLU + Dropout 0.3) → 3 (감성 클래스)
• 입력: 224×224 RGB JPEG, ImageNet 정규화 (mean·std 표준값 사용)
나. 학습 전략 비교: Exp-1 vs Exp-2
실험 Freeze Mode 학습 파라미터 Epoch 의도
Exp-1 fc_only 1,050,627 10 최소 학습 — Backbone 고정
Exp-2 layer4 + fc 16,015,363 7 (Early Stop) 고차 특징 재학습 — 도메인 적응
다. Late Fusion 앙상블; α 가중치 기반 결합
• 결합 방식: P_final = α · P_BERT + (1 - α) · P_ResNet (확률 분포의 가중 평균)
• α 탐색 범위: 0.0 ~ 1.0 (간격 0.05) 검증셋 Macro-F1 최대화
• 선정 사유: 두 모달의 prediction 분포가 상이하여 단순 평균보다 가중 결합이 유효

5. 실험 및 평가 방법 - 11 -
평가 지표 및 실험 설계
가. 평가 지표
지표 정의 채택 사유
Accuracy 전체 정답 / 전체 예측 직관적 성능 척도, 단 클래스 불균형에 민감
Macro-F1 각 클래스 F1의 산술 평균 클래스 불균형 보정(본 연구의 주 지표)
Per-class F1 클래스별 정밀도·재현율의 조화평균 소수 클래스의 학습 효율성 진단
ROUGE-1/2/L Unigram·Bigram·LCS 기반 어휘 중첩
률 KoBART 자동 요약 평가
나. 베이스라인 모델 구성 (단계적 비교)
1) TF-IDF + LinearSVC: 전통 ML 기준선 (CPU 환경 재현 가능)
2) FISA 추론 전용: 도메인 특화 사전학습 모델의 일반화 검증
3) BERT fine-tuning: 본 연구 핵심 텍스트 모델
4) ResNet-50 단독: 이미지 모달 단독 성능 측정
5) Late Fusion 앙상블: 멀티모달 결합의 정량 효과 검증 (가설 H2)
다. 실험 환경 및 재현성
전처리·TF-IDF·FISA·앙상블은 로컬 CPU에서, BERT·ResNet 학습은 RunPod RTX 3090 GPU에서 수행. 모
든 실험에 random_state=42 고정으로 재현성 확보. 결과는 results/scores.json에 일괄 적재.
